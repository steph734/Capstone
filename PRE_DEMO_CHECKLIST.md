## ✨ Demo Flow Quick Reference

**Time: ~5 minutes total**

1. **Login** (30 sec)
   - Show authentication
   - Mention reCAPTCHA security

2. **Quick-Log** (1 min)
   - Click "Show"
   - Select mood emoji
   - Check exercise box → Show progress ring increase
   - Rate pain level
   - Point out "logged successfully"

3. **Calendar** (45 sec)
   - Click "View Calendar"
   - Point to red dot indicators
   - Mention appointment dates

4. **Therapy Categories** (1.5 min)
   - Click therapy card
   - Show modal with exercises/milestones
   - Click "View Full Plan"
   - Navigate to detail page
   - Show progress tracking
   - Click back

5. **Progress Ring** (30 sec)
   - Point to 75% → 80% increase
   - Explain gamification

6. **Notifications** (45 sec)
   - Click bell icon
   - Show notification dropdown
   - Click "Test"
   - Browser notification appears

7. **Responsive** (30 sec - optional)
   - Toggle mobile view
   - Show responsive layout

## 🚨 Emergency Troubleshooting

**If dev server crashes:**
- Have video recording ready as backup
- Restart: `npm run dev`
- Show screenshots while restarting

**If notifications don't work:**
- Explain: "Browser may have blocked permissions"
- Show notification dropdown instead
- Mention: "Works in production environments"

**If internet fails:**
- PWA works offline!
- Mention: "This demonstrates offline capability"
- All features except external APIs work

**If computer freezes:**
- Have backup video on phone
- Or walk through with documentation
